Create a web based app, designed around a pi3 running 32bit Pi OS port of Debian Trixie, using official pi 7" touch screen, switching screen and curser to run in portrait mode, using PAT to connect via blutooth from a phone (for headless operation).

LAYOUT:

--Main menu--
	index.html
		Logo at top
		Buttons (stacked vertically):
			EDID Tools
			Pattern Generator
			Camera
			Input / Output
		Bottom bar:
			Exit browser
			Reboot Pi
			Shutdown Pi
		Footer:
			Version number
			“Update from GitHub” button
			(https://github.com/padge81/AmuseTechTools)

	-Backend logic-
		system.py
			Exit browser
			Reboot
			Shutdown
			Git update
			Version reporting

--EDID tools--
	edid_tools.html
		Home button (top right)
			Controls:
			Read EDID
			Reset
			View toggle (Binary / Decoded)
		Match dialog:
			Displays filename if match found
			Displays “No Match Found” otherwise
		Read window:
			Shows EDID content
		Save feature:
			Disabled if EDID matches existing file
		User supplies filename
			.bin extension enforced
			No overwrite allowed
		Write feature:
			Dropdown of saved .bin files
			Write → Read → Verify
			Results displayed
		Port selector:
			Default port: 2
		Floating status bar:
			Read / Write / Verify / Error states
	-Backend logic-
		edid.py
			Read EDID
			Compare EDID
			Decode EDID
			Save EDID
			Write + verify EDID

--Pattern generator--
	pattern_generator.html
		Home button (top right)
		Under construction image
	-Backend-
		pattern_generator.py (placeholder)

--Camera--
	camera.html
		Home button (top right)
		Under construction image
	-Backend-
		camera.py (placeholder)

--Input / Output hub--
	input_output.html
		Home button (top right)
		Buttons stacked vertically:
			Basic IO
			Ticket Dispenser
			Coin Mech
			Card Reader
			Hopper
	-Backend-
		input_output.py (navigation only)

--Submodules--
Each submodule follows the same pattern:
	basic_io.html
	Home button (top right)
	Back button (top left)
	Under construction image

--Backend--
	basic_io.py
Same structure for:
	ticket_dispenser
	coin_mech
	card_reader
	hopper
